# Задание на зачетную работу

Создание проекта печатной платы для МК-системы на основе МК Atmega8515.
Разработать таймерное устройство, построенное на микроконтроллерах, регистрирующее в течении заданного времени (1, 2 или 3 часа) работу 3-х таймеров. Работа всех 3-х таймеров начинается одновременно, а прекращается в каждом таймере независимо друг от друга по сигналу останова. По истечении заданного интервала времени работа устройства прекращается. Подсчитать суммарное время работы таймеров. По запросу с пульта оператора вывести индикацию последовательно время работы таймера 1 и его процедурную долю от общего времени работы таймеров, затем для таймера 2 и таймера 3. 
Предусмотреть передачу результатов на компьютер по интерфейсу RS-232 по запросу от компьютера. 
Разработать схемы, алгоритмы и программы. Для отладки программ использовать виртуальную системы моделирования VSM из пакета PROTEUS. При отладке программы использовать виртуальные внешние устройства: кнопки, дисплей на 7-сегментных индикаторах или ЖКД для отображения времени
Тип микроконтроллера - ATmega8515.
